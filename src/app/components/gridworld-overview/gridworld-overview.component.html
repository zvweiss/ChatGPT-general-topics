Gridworld: Overview

The Gridworld environment is a classic example from Sutton and Barto's reinforcement learning book. It's a grid of states where an agent can move in four directions, receive rewards, and learn value estimates over time.

This page introduces the purpose of the environment, and outlines the internal structure of the implementation.

🗂 File Structure


src/
├── app/
│   ├── components/
│   │   ├── gridworld/             ← GridworldComponent (main view + controls)
│   │   ├── grid-cell/             ← GridCellComponent (renders each cell)
│   │   └── gridworld-overview/    ← This overview page
│   ├── services/
│   │   └── gridworld.service.ts   ← Manages grid state, agent, and movement
│   ├── models/
│   │   └── gridworld-model.ts     ← Types for cells, agent, directions

📘 Demonstration Goals

  • Visualize state space and agent movement
  • Introduce value functions and learning updates
  • Show how RL agents improve over time
  • Bridge theory with hands-on interaction
